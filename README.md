# AutoCarePro
A professional, fully-responsive car workshop website built with HTML5, CSS3, and JavaScript.
## 🚗 Features

- **6 Pages**: Home, Services, Mechanics, Gallery, Booking, Contact
- **Interactive**: Service selection, cost calculator, form validation, filtering
- **Responsive**: Works perfectly on mobile and desktop
- **Modern UI**: Automotive theme with smooth animations

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Modern styling with animations
- **JavaScript** - Interactive functionality
- **Font Awesome** - Icons

## 📁 Files

```
AutoCare-Pro/
├── index.html          # Home page
├── services.html       # Services page
├── mechanics.html      # Mechanics page
├── gallery.html        # Gallery page
├── booking.html        # Booking page
├── contact.html        # Contact page
├── styles.css          # Complete styling
├── script.js           # All functionality
```

## 🚀 Quick Start

1. Open `index.html` in your web browser
2. Navigate through all pages
3. Test the booking system and service selection

## 📱 Responsive Design

- Mobile-first approach
- Tablet and desktop optimized
